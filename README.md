## 阅读笔记

**贝叶斯法则展示了用于接受新的数据和重新计算概率估计的数学上的正确方法。**

### 1. 贝叶斯法则，先验以及后验

这里先给出贝叶斯法则：

$$
P(A|B)=P(B|A)\cfrac{P(A)}{P(B)}
$$

从右向左：

* $P(A)$是没有新数据时假设的概率，称之为**先验**；
* $P(B)$是看见新数据的总体概率，称之为**边际似然**；
* $P(B|A)$是给定最初假设时看见新数据的概率，称之为**似然**；
* $P(A|B)$是接受了新数据之后重新预估的概率，称之为**后验**；

也就是说，贝叶斯法则给出了通过先验计算后验的一种方法。
