#### 网格坐标

通过itertools可以快速生成网格坐标，一段示例程序如下：

```python
>>> import itertools
>>> x = [1, 2]
>>> y = [4, 5]
>>> xAndy = list(itertools.product(x, y))
>>> xAndy
[(1, 4), (1, 5), (2, 4), (2, 5)]
```

#### 退出程序

向sys.exit()传入参数1可以让程序因错误而终止，对应的，可以在之前相关的print函数中指定file=sys.stderr，这样在解释器中print函数打印出来的“错误信息”就是红色的了。
